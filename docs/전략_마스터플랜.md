## 2. 현실적 목표 (수학적으로 검증됨)

### Phase 1 목표: 월 $2,000-3,000 (사이드잡)

```
목표: 월 $2,000 (제휴 리커링)
계산:
  평균 커미션: 30% × $100/월 SaaS = $30/건/월
  필요 유료 전환: $2,000 ÷ $30 = 67명
  필요 클릭 (전환율 2%): 67 ÷ 0.02 = 3,350 클릭
  필요 방문자 (CTR 10%): 3,350 ÷ 0.10 = 33,500명/월

→ 월 33,500명 방문자 = 12-18개월 pSEO로 달성 가능 ✅
```

### Phase 2 목표: 월 $7,500 (1,000만원) — 확장 후

```
조건: Phase 1 달성 후 (트래픽 3-5만명 확보된 상태)
추가 수익원:
  유료 도구 등록  $2,000  ← 트래픽 있으면 영업 가능
  뉴스레터 스폰서 $2,000  ← 구독자 5K+ 달성 후
  Featured 리스팅 $1,500  ← 트래픽 증명 후
  제휴 커미션     $1,000  ← Phase 1 기반
  광고 수익       $  500  ← 50K 세션 달성 후
  ────────────────────────
  합계            $7,000+
```

**핵심 순서: 트래픽 먼저 → B2B 수익 나중**
트래픽 0명일 때 유료 등록 영업하면 무시당함.
트래픽 3-5만명 생기면 AI 도구 회사가 먼저 연락옴.

### 수익 타임라인

| 기간 | 월 수익 | 핵심 마일스톤 |
|------|---------|-------------|
| 1-6개월 | $0-500 | pSEO 구축 완성, 인덱싱, 첫 트래픽 |
| 7-12개월 | $500-2,000 | 롱테일 랭킹 시작, 제휴 전환 |
| 13-18개월 | $2,000-3,000 | **Phase 1 목표 달성** |
| 19-24개월 | $5,000-7,500 | B2B 수익 추가, **월천 달성** |

## 3. 우리의 강점 & 차별화 포인트

### 현재 실제로 있는 강점

| 강점 | 현황 | 경쟁사 대비 |
|------|------|-----------|
| **비교 페이지 1,526개** | 구축 완료 ✅ | TAAFT·Toolify 비교 기능 약함 |
| **현대적 기술 스택** | Next.js 15 SSG | 경쟁사 대부분 WordPress → 속도 느림 |
| **가격 비교 페이지** | /pricing/[slug] | 경쟁사 대부분 가격 정보 단순 나열 |
| **대안 페이지** | /alternatives/[slug] | 구매 의도 높은데 경쟁 상대적으로 적음 |
| **후발 주자 이점** | 경쟁사 실수 분석 완료 | 처음부터 더 잘 만들 수 있음 |

### 공략할 틈새 (경쟁사가 못하는 것)

```
TAAFT     → 규모 크지만 비교 기능 없음, UI 단순
Toolify   → 도구 많지만 편집 품질 낮음 (데이터 덤프 느낌)
Futurepedia → 품질 있지만 pSEO 활용 부족
FutureTools → 개인 브랜드(유튜버) 의존, 확장성 한계

AIToolRadar의 포지션:
"pSEO 규모 + 비교 깊이 + 콘텐츠 품질"
이 세 가지를 동시에 잘하는 사이트가 없음
```

### 콘텐츠 차별화 전략 (content_improvement_plan 요약)
- 189개 도구를 5가지 패턴으로 분산 (구글 스팸 필터 회피)
- E-E-A-T 강화 문구 삽입 ("In our testing...", "What stands out...")
- 도구 인지도별 콘텐츠 길이 차등 적용
- 반복 구조 탈피 → 신뢰도 + 랭킹 향상

## 8. 절대 하지 말아야 할 것 (원칙)

```
❌ 트래픽 없을 때 B2B 영업 (유료 등록, 스폰서)
   → 시간 낭비, 무시당함. 트래픽 생기면 자연히 연락옴.

❌ 두 번째 사이트 시작
   → Phase 1 달성 전까지 절대 금지. 분산되면 둘 다 실패.

❌ 빈 페이지 양산 (content_html 없는 상태로 페이지만 늘리기)
   → 구글 스팸 판정 위험. 품질 먼저, 양은 나중.

❌ 완벽주의
   → 80% 완성으로 빨리 배포하고 데이터로 개선. 완벽함보다 속도.

❌ 제휴 링크만으로 월 $7,500 목표 (수학적으로 불가능)
   → 제휴는 Phase 1 ($2-3K)까지. 그 이상은 B2B 추가 필요.
```

## 9. 우리의 핵심 원칙 5가지

1. **자동화 우선**: 한 번 만들면 자동으로 돌아가는 구조. 반복 노동 최소화.
2. **순서를 지킨다**: 트래픽 먼저 → 수익 다각화 나중. 순서 바꾸면 실패.
3. **한 사이트에 올인**: 월 $3,000 달성 전까지 다른 프로젝트 없음.
4. **품질이 양보다 먼저**: 177개 도구 고품질 > 500개 빈 페이지.
5. **데이터로 결정한다**: 감이 아닌 GA4·Search Console 데이터 기반 의사결정.
