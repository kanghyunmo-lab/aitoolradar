## 3. content_html 5가지 패턴 (E-E-A-T 가이드)

> Antigravity로 도구 확장 시 이 패턴을 사용해야 구글 스팸 필터 회피 가능.

### 패턴 분배 비율
| 패턴 | 비율 | 대상 |
|------|------|------|
| A. Editor's Review | 40% | 중간 인지도 도구 |
| B. Use Case Focus | 20% | 특정 직업군 타겟 도구 |
| C. Comparison Angle | 15% | 경쟁 도구 있는 경우 |
| D. Deep Dive | 15% | 인기 도구 (ChatGPT, Jasper 등) |
| E. Quick Take | 10% | 니치/단순 도구 |

### 패턴 A — Editor's Review (가장 많이 사용)
```html
<h2>What Is [Tool]?</h2>
<p>Tool description... "What stands out when you actually use it is..."</p>
<h2>What We Like</h2>
<p>"In our experience, the [feature] works best for..."</p>
<h2>What Could Be Better</h2>
<p>"The learning curve is steeper than expected..."</p>
<h2>Pricing</h2>
<p>Pricing details...</p>
```

### 패턴 B — Use Case Focus
```html
<h2>[Tool]: [One-line value proposition]</h2>
<p>Overview focused on the problem it solves...</p>
<h2>Best For</h2>
<p>"If you are a [specific user], [Tool] is worth considering because..."</p>
<h2>Key Capabilities</h2>
<ul><li>Feature + benefit</li></ul>
<h2>How Much Does It Cost?</h2>
<p>Pricing...</p>
```

### 패턴 C — Comparison Angle
```html
<h2>Why Consider [Tool]?</h2>
<p>What makes it different from competitors...</p>
<h2>Standout Features</h2>
<p>What this tool does that others don't...</p>
<h2>Alternatives Worth Exploring</h2>
<p>"If [Tool] feels too enterprise-heavy, [Alternative] offers..."</p>
<h2>Pricing Overview</h2>
<p>Pricing...</p>
```

### 패턴 D — Deep Dive (인기 도구용)
```html
<h2>[Tool] in 2026: Is It Worth It?</h2>
<p>Current assessment...</p>
<h2>The Good</h2>
<p>"After spending time with [Tool], the feature that impressed us most was..."</p>
<h2>The Not-So-Good</h2>
<p>"Where [Tool] falls short is..."</p>
<h2>Who Should Use [Tool]?</h2>
<p>Specific target users...</p>
<h2>Pricing Breakdown</h2>
<p>Detailed pricing...</p>
```

### 패턴 E — Quick Take (단순 도구용)
```html
<h2>Quick Take: [Tool]</h2>
<p>2~3 sentence core summary...</p>
<h2>What It Does</h2>
<p>Concise feature overview...</p>
<h2>Pricing</h2>
<p>Pricing...</p>
```

### E-E-A-T 강화 필수 문구 (영어)
| 카테고리 | 문구 |
|----------|------|
| 경험 시그널 | "In our testing...", "After spending time with...", "What stands out when you actually use it..." |
| 비교 맥락 | "Unlike [X], which focuses on...", "Compared to [X], this tool..." |
| 구체적 수치 | "generates 1,500 words in under 2 minutes", "supports 25+ languages" |
| 균형 평가 | "works well for X but struggles with Y", "the free plan is generous, though..." |
| 대안 언급 | "If [Tool] is too expensive, consider [Alternative]" |

### 콘텐츠 길이 기준
| 도구 유형 | 영어 단어 수 | 패턴 |
|-----------|------------|------|
| 대중 인기 도구 (ChatGPT, Jasper) | 250~350 words | D |
| 중간 인지도 | 150~250 words | A/B/C |
| 니치/단순 | 100~150 words | E |

## 4. 수익 수학 (참고용)

```
리커링 제휴만으로 월 $2,000 달성 계산:
  평균 커미션: 30% × $100/월 SaaS = $30/건/월
  필요 유료 전환: $2,000 ÷ $30 = 67명
  필요 클릭 (전환율 2%): 67 ÷ 0.02 = 3,350 클릭/월
  필요 방문자 (CTR 10%): 3,350 ÷ 0.10 = 33,500명/월

→ 월 3~5만명 방문자 = 12~18개월 pSEO로 달성 가능 (AI Scout 사례 검증됨)
```
