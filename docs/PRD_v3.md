## 3. 확정 전략 (3단계 로드맵)

### Stage 1 — 제휴 링크 CTA 전면 반영 (이번 주)
> "트래픽 1명이라도 들어왔을 때 돈이 될 구멍부터"

**직접:**
- Writesonic 승인 확인 → 링크 복사
- Semrush 가입 (semrush.com/partners)
- DeepL 가입 (PartnerStack)
- Copy.ai 가입 (PartnerStack)
- Jasper 가입 재시도 (PartnerStack)

**Claude Code (링크 받는 즉시):**
- 신규 도구 DB 등록 (Writesonic, DeepL, Synthesia, Frase.io)
- affiliate_url DB 업데이트 SQL
- CTA 버튼 4개 페이지 타입 전면 연결 (리뷰/비교/가격/대안)
- GA4 affiliate_click 이벤트 연결

---

### Stage 2 — 도구 수 확장 300~500개 (다음 달)
> "구글에게 우리가 권위자임을 선포"

- Antigravity로 도구 300~500개로 확장
- content_html 품질 기준 유지 (5가지 패턴 적용, 참고: `참고자료.md`)
- 신규 도구 추가 시 비교 페이지도 자동 생성됨

---

### Stage 3 — AI 도구 추천 퀴즈 (Stage 1 완료 후, Claude Code)
> "나에게 맞는 AI 도구 찾기 — 수익형 깔때기"

```
유저: "나는 마케터야, 매일 SNS 글을 써야 해"
  ↓ 퀴즈 3~5개 질문 (용도 / 예산 / 기술 수준)
  ↓ 결과: "Writesonic이 최적입니다"
  ↓ CTA: "Try Writesonic Free" ← 제휴 링크 → 수익
```

## 5. 수익 모델 (확정)

**Phase 1 목표: 월 $500~1,500 (6~12개월)**
```
리커링 제휴 커미션만으로:
  평균 커미션 $30/건/월
  월 20명 신규 전환 → 6개월 누적 120명 × $30 = $3,600/월
  필요 방문자: 월 2~5만명 (12~18개월 pSEO로 달성 가능)
```

**Phase 2 목표: 월 $3,000~7,500 (18~24개월)**
```
트래픽 3~5만명 확보 후:
  제휴 리커링   $1,000~2,000
  B2B 유료 등록 $2,000       ← 트래픽 있으면 영업 가능
  Featured 리스팅 $1,500     ← 트래픽 증명 후
  뉴스레터 스폰서 $2,000     ← 구독자 5K+ 후
```

## 7. 핵심 원칙 5가지

1. **순서를 지킨다**: 트래픽 먼저 → B2B 수익 나중. 순서 바꾸면 실패.
2. **자동화 우선**: 한 번 만들면 자동으로 돌아가는 구조. 반복 노동 최소화.
3. **품질이 양보다 먼저**: 빈 페이지 양산은 구글 스팸 판정 위험.
4. **한 사이트에 올인**: 월 $3,000 달성 전까지 다른 프로젝트 없음.
5. **데이터로 결정한다**: 감이 아닌 GA4·Search Console 기반 의사결정.
